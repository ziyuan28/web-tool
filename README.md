# 工具箱 

一款集成多种实用功能的单页Web应用，包含代办清单、番茄钟、便签、密码生成、倒计时和记账本等工具，界面美观且响应式设计适配各种设备。

## 功能特点

- **代办清单**：快速添加、标记完成和删除任务，自动本地保存
- **番茄专注**：自定义时长的番茄钟，帮助提高工作效率
- **随手便签**：简单的文本记录工具，自动保存内容
- **密码生成**：可自定义长度和字符类型的随机密码生成器
- **倒计时**：设置目标日期，实时显示剩余时间
- **记账本**：记录收入和支出，自动计算总额

## 技术栈

- HTML5 + CSS3
- Tailwind CSS v3：用于样式设计
- Vue 3：用于前端交互逻辑
- Font Awesome：提供图标支持
- 本地存储(localStorage)：用于数据持久化

## 界面预览

应用采用现代化UI设计，包含左侧导航栏和主内容区：
- 左侧导航：可折叠式设计，在移动设备上自动适应
- 主内容区：根据选择的工具动态切换内容，带有平滑过渡动画
- 响应式布局：完美适配从手机到桌面的各种屏幕尺寸

## 使用方法

1. 直接打开`index.html`文件即可使用，无需额外部署
2. 所有数据保存在浏览器本地存储中，刷新页面不会丢失
3. 点击左侧导航图标切换不同工具
4. 各工具使用说明：
   - 代办清单：输入任务后按回车或点击"+"添加
   - 番茄钟：可选择预设时长或自定义时间，点击播放按钮开始
   - 便签：直接输入内容即可自动保存
   - 密码生成：调整参数后点击"生成随机密码"，点击复制图标复制
   - 倒计时：输入标题和日期，点击"保存"开始倒计时
   - 记账本：选择类型，输入项目和金额，点击"√"添加记录

## 自定义与扩展

如需添加新功能，可按照现有模式：
1. 在`navItems`数组中添加新工具的导航信息
2. 在主内容区的transition中添加新工具的HTML结构
3. 在Vue实例的setup函数中添加相应的数据和方法
4. 实现本地存储逻辑以保持数据持久化
